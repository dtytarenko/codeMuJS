Задачи для решения
Работа с переменными

1. Создайте переменную num и присвойте ей значение 3. Выведите значение этой переменной на экран с помощью метода alert. Показать решение.

2. Создайте переменные a=10 и b=2. Выведите на экран их сумму, разность, произведение и частное (результат деления). Показать решение.

3. Создайте переменные c=15 и d=2. Просуммируйте их, а результат присвойте переменной result. Выведите на экран значение переменной result. Показать решение.

4. Создайте переменные a=10, b=2 и c=5. Выведите на экран их сумму. Показать решение.

5. Создайте переменные a=17 и b=10. Отнимите от a переменную b и результат присвойте переменной c. Затем создайте переменную d, присвойте ей значение 7. Сложите переменные c и d, а результат запишите в переменную result. Выведите на экран значение переменной result. Показать решение.

Создайте переменную name и присвойте ей ваше имя. Выведите на экран фразу 'Привет, %Имя%!'. Показать решение.

Создайте переменную age и присвойте ей ваш возраст. Выведите на экран 'Мне %Возраст% лет!'. Показать решение.

Создайте переменную num и присвойте ей значение '12345'. Найдите произведение (умножение) цифр этого числа. Показать решение.

Работа с присваиванием и декрементами

Переделайте этот код так, чтобы в нем использовались операции +=, -=, *=, /=. Количество строк кода при этом не должно измениться.

var num = 47;
num = num + 7;
num = num - 18;
num = num * 10;
num = num / 15;
alert(num);

Показать решение.
