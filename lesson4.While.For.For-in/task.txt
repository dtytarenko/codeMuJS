Задачи для решения
Циклы while и for
Решите эти задачи сначала через цикл while, а затем через цикл for.
 Выведите столбец чисел от 1 до 100. Показать решение.

 Выведите столбец чисел от 11 до 33. Показать решение.

 Выведите столбец четных чисел в промежутке от 0 до 100. Показать решение.

 С помощью цикла найдите сумму чисел от 1 до 100. Показать решение.

Работа с for для массивов
 Дан массив с элементами [1, 2, 3, 4, 5]. С помощью цикла for выведите все эти элементы на экран. Показать решение.

 Дан массив с элементами [1, 2, 3, 4, 5]. С помощью цикла for найдите сумму элементов этого массива. Запишите ее в переменную result. Показать решение.

Работа с for-in
 Дан объект obj. С помощью цикла for-in выведите на экран ключи и элементы этого объекта.

	var obj = {green: 'зеленый', red: 'красный', blue: 'голубой'}
Показать решение.

 Дан объект obj с ключами Коля, Вася, Петя с элементами '200', '300', '400'. С помощью цикла for-in выведите на экран строки такого формата: 'Коля - зарплата 200 долларов.'. Показать решение.

Задачи
 Дан массив с элементами 2, 5, 9, 15, 0, 4. С помощью цикла for и оператора if выведите на экран столбец тех элементов массива, которые больше 3-х, но меньше 10. Показать решение.

 Дан массив с числами. Числа могут быть положительными и отрицательными. Найдите сумму положительных элементов массива. Показать решение.

 Дан массив с элементами 1, 2, 5, 9, 4, 13, 4, 10. С помощью цикла for и оператора if проверьте есть ли в массиве элемент со значением, равным 4. Если есть - выведите на экран 'Есть!' и выйдите из цикла. Если нет - ничего делать не надо. Показать решение.

 Дан массив числами, например: [10, 20, 30, 50, 235, 3000]. Выведите на экран только те числа из массива, которые начинаются на цифру 1, 2 или 5. Показать решение.

 Дан массив с элементами 1, 2, 3, 4, 5, 6, 7, 8, 9. С помощью цикла for создайте строку '-1-2-3-4-5-6-7-8-9-'. Показать решение.

 Составьте массив дней недели. С помощью цикла for выведите все дни недели, а выходные дни выведите жирным. Показать решение.

 Составьте массив дней недели. С помощью цикла for выведите все дни недели, а текущий день выведите курсивом. Текущий день должен храниться в переменной day. Показать решение.

 Дано число n=1000. Делите его на 2 столько раз, пока результат деления не станет меньше 50. Какое число получится? Посчитайте количество итераций, необходимых для этого (итерация - это проход цикла), и запишите его в переменную num. Показать решение.
